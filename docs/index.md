## 关于本项目

本项目包含以下几个模块：

### 1️⃣《Java小白求职之路》

目标人群为 即将找工作 + 初级开发 的程序员。

收集了醋酸菌本人面试Java岗位过程中遇到的题目。包含三个档位薪资的题目：

- 初级 8~10k
